# Acel AOV API

![](/static/assets/banners/whg_acelapi.png)

!!!danger Important Notice
This product has been discontinued and is no longer supported. The information provided may be incomplete or outdated. Please refrain from contacting support regarding this product, as dedicated assistance is no longer available.
!!!
!!!info
This is a technical page, and assumes familiarity with Roblox Studio's Explorer, alongside with Roblox's scripting language, Luau.
!!!

---

Located directly in the root folder **after game startup**, the API allows you to control the windows, making script automation possible, and you won't have to rely on the Controller.

## Syntax

```lua
workspace["Acel AOV System"].AcelCmd:Fire(Event)
```
!!!light What do these mean?
**Event**: (`string`) Function for the windows to execute.
!!!
  Acel's function list is as follows:
- `"Open"`: Opens the windows.
- `"Close"`: Closes the widows.
